# AUS: Assistente Universal de Saúde (development version)

O AUS será um aplicativo na área de saúde integrado ao SUS...

### Funcionalidades

- Ficha Médica
  - Condições médicas
    - Doenças diagnosticadas
    - Predisposições
    - Condições psiquiátricas
    - Alergias e reações
  - Histórico
    - Medicamentos
    - Cirurgias
    - Exames
  - Dados do paciente
    - Fixos
      - Tipo sanguíneo
      - Sexo
      - Tipo de pele Fitzpatrick
    - Variáveis
      - Peso
      - Altura
      - Gordura corporal
      - Grau (visão)
- Integração com dispositivos inteligentes
  - SmartWatch
  - Smart Shoes
  - Fitbit (Pulseiras)
  - Balança de Bioimpedância
  - Medidor de glicose
  - Medidor de pressão
  - Oximetro
  - Termômetro
  - Aparelhos de academia
- Integração com o SUS
  - Plataforma do médico
